# Productivity Tracker

CMPA4403 — Semester Project

---

## What is this?

Productivity Tracker is a lightweight, browser-based daily task manager. It lets you add tasks grouped by category (School, Work, or Personal), check them off as you complete them, and track your progress with a live stats bar showing total, completed, and remaining tasks. The interface includes a dark mode toggle, colour-coded category badges, and smooth hover animations — all without any frameworks or installs required.

---

## Why does it exist?

The goal was to build a practical tool that solves a real everyday problem: keeping track of what needs to get done across different areas of life. Instead of mixing school assignments, work tasks, and personal errands into one undifferentiated list, this tracker lets you separate and filter them at a glance. It was also designed to be as frictionless as possible — no sign-in, no setup, just open and start adding tasks.

---

## What tools did I use?

| Tool | Purpose |
|---|---|
| **HTML5** | Page structure and semantic markup |
| **CSS3** (Flexbox & Grid) | Layout, responsive design, and all visual styling including dark mode via CSS custom properties |
| **Vanilla JavaScript** | Task logic, localStorage persistence, dark mode toggle, and category filtering |
| **localStorage API** | Saves tasks and theme preference in the browser between visits — no backend needed |
| **GitHub Pages** | Free static site hosting with automatic deployment on every push to `main` |
No external libraries or frameworks were used — the project runs entirely on native browser APIs.
