A little background on this project

What is this?

Productivity Tracker is a lightweight, browser-based task manager built as a semester project for CMPA4403. It lets you log your daily tasks, check them off as you complete them, and keep a clear picture of what still needs to get done — all without signing in or installing anything.

Tasks are saved in your browser's local storage, so they persist between visits on the same device.

Tools used

HTML5 CSS3 Flexbox & Grid Vanilla JavaScript localStorage API GitHub Pages

Features

Add tasks with a single keystroke, check them off, and remove them when they're no longer relevant. A live stats bar at the top of the home page shows your total, completed, and remaining task count at a glance. The layout is fully responsive and works on mobile, tablet, and desktop.

Challenges

The trickiest part was making the task list feel snappy without a JavaScript framework — re-rendering the list on every state change while keeping the DOM tidy took a few iterations. Getting the responsive grid to collapse gracefully on small screens also required some careful testing across device sizes.
