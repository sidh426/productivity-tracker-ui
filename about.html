A little background on this project

What is this?

Productivity Tracker is a lightweight, browser-based task manager built as a semester project for CMPA4403. It lets you log your daily tasks by category, check them off as you complete them, and keep a clear picture of what still needs to get done — all without signing in or installing anything.

Tasks are saved in your browser's local storage, so they persist between visits on the same device.

Tools used

HTML5 CSS3 Flexbox & Grid Vanilla JavaScript localStorage API GitHub Pages

Features

Add tasks with a single keystroke and assign them to a category — School, Work, or Personal — each colour-coded for instant recognition. Filter your list by category, check tasks off as you go, and remove them when they're no longer relevant. A live stats bar tracks your total, completed, and remaining task count. Dark mode is supported and remembers your preference between visits.

Challenges

The trickiest part was making the task list feel snappy without a JavaScript framework — re-rendering on every state change while keeping the DOM tidy took a few iterations. Adding a filter layer on top of that without breaking index references into the underlying array required careful tracking of each task's real position. Preventing a flash of unstyled content when switching between dark and light mode also required an inline script in the document head to apply the saved theme before the first paint.
